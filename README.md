### Usage

#### Direct Use

1. Open the [download page](https://github.com/LoveProgrammingMint/Xdows-Security/releases ) to download the latest version of `Xdows-Security.zip`
2. Extract the zip file to the target location and run `Xdows-Security.exe`

#### Compile and Run

1. Environment requirements:
    1. Windows 10/11
    2. Install VS2026 or higher and related workloads
    3. Install Git and ensure normal access to GitHub.
2. Clone the repository
```sh
git clone https://github.com/LoveProgrammingMint/Xdows-Security 
git clone https://github.com/LoveProgrammingMint/SouXiaoAVEngine
```
3. Open with VS and build
